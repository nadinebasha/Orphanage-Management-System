# Orphanage Management System

A full-stack management platform designed to support orphanage operations, donation tracking, and adoption workflows.

---

## Features
- Donation management
- Adoption request workflows
- Guardian and child record management
- Database-driven backend system
- MVC-based architecture

---

## Technologies Used
- PHP
- MySQL
- JavaScript
- HTML
- CSS
- MVC Architecture

## Functionalities
- Manage orphan records
- Process donations
